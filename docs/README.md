## üîç Quick Reference

### Essential Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+K` | Open Command Palette |
| `Ctrl+Shift+T` | Toggle Theme |
| `Ctrl+Shift+S` | Create Backup |
| `Ctrl+/` | Open Helper AI |
| `Ctrl+Alt+4` | Quad Layout |

Full list: [KEYBOARD_SHORTCUTS.md](KEYBOARD_SHORTCUTS.md) (Coming Soon)

### Essential Commands

```bash
# System status
va21-cli status

# Create backup
va21-cli backup create

# Restore backup
va21-cli backup restore --from <backup-id>

# Search vault
va21-cli vault search "query"

# Install app
va21-cli app install <app-name>
```

### Helper AI Quick Commands

```
"create backup" - Backup system now
"restore from yesterday" - Restore previous backup
"system status" - Check system resources
"install <app>" - Install application
"search vault for <query>" - Search knowledge vault
```
